# 📊 Meta Ads Performance Dashboard | Power BI Project

## 📌 Project Overview
This project is a **Meta (Facebook & Instagram) Ads Performance Dashboard** built using **Power BI** to analyze digital marketing campaign performance.

The dashboard helps marketing teams and analysts track key metrics such as impressions, clicks, engagement, CTR, and conversion rates to understand how advertising campaigns are performing.

It provides an **interactive and visual way to monitor marketing performance and make data-driven decisions.**

---

## 🎯 Business Problem
Digital marketing teams run multiple ad campaigns across platforms like **Facebook and Instagram**.  
However, analyzing campaign performance from raw data can be difficult.

This dashboard solves that problem by:

- Tracking **campaign performance metrics**
- Identifying **high performing ads**
- Understanding **user engagement behavior**
- Monitoring **budget and conversions**

---

## 📈 Dashboard Features

### 1️⃣ KPI Performance Cards
Quick overview of important marketing metrics like CTR, conversions, and engagements.

### 2️⃣ Engagement Analysis
User engagement breakdown by:
- Gender
- Age group

### 3️⃣ Geographic Analysis
Map visualization showing **engagement distribution across regions**.

### 4️⃣ Time-Based Analysis
- Weekly engagement trends
- Hourly engagement patterns
- Monthly campaign analysis

### 5️⃣ Ad Type Performance
Comparison of different ad formats:

- Carousel Ads
- Image Ads
- Stories Ads
- Video Ads

Metrics analyzed include:

- Impressions
- Clicks
- CTR
- Engagement Rate
- Conversion Rate

### 6️⃣ Interactive Filters
Users can dynamically filter dashboard by:

- Campaign Name
- Target Interest
- Performance Metrics

- Engagement peaks during specific hours of the day
